- Publish launch assets
- Drive first qualified traffic
- Close first sales

## Hour-by-Hour Runbook

## T-2:00 (Prep Block)
- [ ] Confirm product files are accessible and download links work.
- [ ] Confirm price state: intro ($79) active and cap tracking in place.
- [ ] Confirm checkout test purchase done end-to-end.
- [ ] Confirm legal pages linked (terms/privacy/disclosure).
- [ ] Set UTM conventions for all channels:
  - `utm_source` (x, email, dm, affiliate, youtube)
  - `utm_medium` (organic, outbound, partner)
  - `utm_campaign=overnight_launch_kit_launch_week`

## T-1:00 (Publishing Setup)
- [ ] Load short post queue (minimum 4 posts ready).
- [ ] Load long thread #1 draft.
- [ ] Prepare email #1 with links + fallback plain text version.
- [ ] Prepare DM list (40–60 targets segmented: operators/affiliates).
- [ ] Open tracking sheet (`docs/FUNNEL_TRACKING_SHEET_TEMPLATE.csv`).

## H0 (Launch Moment)
- [ ] Publish hero launch post on X.
- [ ] Publish thread #1 (value-first, CTA at end).
- [ ] Send email #1 to warm list.
- [ ] Send first 20 DMs (high-intent targets first).
- [ ] Log first 30-minute metrics snapshot.

## H+1
- [ ] Reply to all post comments and DMs within 10 minutes.
- [ ] Send next 20 DMs.
- [ ] Check checkout starts and purchases.
- [ ] If zero checkout starts: verify links and checkout uptime immediately.

## H+2
- [ ] Publish short post #2 (pain/cost angle).
- [ ] Publish YouTube short #1.
- [ ] Push partner outreach batch #1 (5–10 affiliates/operators).
- [ ] Update tracking sheet with source-level numbers.

## H+3
- [ ] Review top traffic source by CTR.
- [ ] Duplicate winning hook into a new short post draft.
- [ ] Send follow-up to non-responding high-value DMs from H0.

## H+4
- [ ] Publish short post #3 (objection handling).
- [ ] Send second email segment or resend to non-openers (if list size supports it).
- [ ] Share mini-proof update (e.g., “X downloads/sales in first hours” if true).

## H+5
- [ ] Partner check-ins: confirm who posted, who needs copy/link help.
- [ ] Send DM wave #3 (remaining list).
- [ ] Evaluate landing page friction (scroll depth, CTA clicks, checkout starts).

## H+6
- [ ] Publish short post #4 (urgency + intro cap reminder).
- [ ] Respond to objections in public (pricing, fit, implementation time).
- [ ] Log midpoint KPI snapshot and compare vs target.

## H+7
- [ ] If below target: deploy contingency
  - [ ] Add stronger CTA above fold
  - [ ] Add clearer deliverables bullets
  - [ ] Add FAQ snippet to reduce hesitation
- [ ] If above target: increase urgency posts and partner reminders.

## H+8
- [ ] Publish “operator tip” post with soft CTA.
- [ ] Send follow-up DM to all clickers who didn’t buy.
- [ ] Refresh leaderboard of partner clicks/sales.

## H+9
- [ ] Publish YouTube short #2 or repurposed clip.
- [ ] Run a live Q&A thread/replies for objections.
- [ ] Check refund/complaint signals (if any).

## H+10
- [ ] Publish end-of-day urgency post.
- [ ] Send “last call today” message to warm DM responders.
- [ ] Snapshot full funnel metrics by source.

## H+11
- [ ] Compile end-of-day performance report:
  - Visits, CTR, checkout starts, purchases, refunds, CVR, CAC (if paid)
- [ ] Document top 3 winners and top 3 frictions.
- [ ] Queue next-day creative based on winning hook.

## H+12 (Closeout)
- [ ] Update proof log with what is now validated by data.
- [ ] Confirm whether intro cap is near/at limit.
- [ ] Set day-2 plan (what to double down on, what to cut).

  - [ ] Refund/quality complaints start early
