# Copy Preflight Checklist (Approval Before Send/Publish)

Use this quick checklist before any email, ad, SMS, landing page, or social post goes live.

## 1) Blocked Phrases Check
- [ ] I reviewed the copy against the blocked phrases list.
- [ ] No blocked phrase appears in headline, body, CTA, footer, image text, or metadata.
- [ ] If uncertain wording exists, it was replaced or escalated for compliance review.

## 2) Mandatory Footer Fields Check
- [ ] Brand/company legal name is present.
- [ ] Valid physical mailing address is present.
- [ ] Contact method is present (support email and/or phone).
- [ ] Required disclosure/compliance text is present (channel-specific).
- [ ] Unsubscribe/opt-out instructions are present and clear (where required).

## 3) CTA Allowlist Check
- [ ] Every CTA used is on the approved CTA allowlist.
- [ ] CTA wording exactly matches an allowlisted option (no unapproved variants).
- [ ] CTA destination/URL matches the approved route/offer.

## 4) Single-CTA Rule Check
- [ ] Only one primary CTA is present in the final copy.
- [ ] No competing secondary CTAs are included.
- [ ] Buttons, links, and anchor text all point to the same single CTA intent.
