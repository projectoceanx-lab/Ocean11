# Email Sequence Blocks

Use one objective per email. Keep one primary CTA.

## Block A: Acknowledge + Clarify + CTA
- Subject: `{{first_name}}, review your available options`
- Body:
  1. Acknowledge the user context.
  2. Clarify what they can do next without promises.
  3. Present one CTA.
- CTA: `See If You Qualify`

## Block B: Education + Low-friction Step
- Subject: `A quick way to compare debt options`
- Body:
  1. Explain one practical concept (for example, unsecured debt eligibility).
  2. Encourage a single next step.
  3. Keep certainty language soft ("may", "could", "results vary").
- CTA: `Check Your Options`

## Block C: Re-engagement
- Subject: `Your options are still available`
- Body:
  1. Brief reminder.
  2. Explain value of checking now vs waiting.
  3. One CTA only.
- CTA: `View Matched Options`

## Required Footer (Debt Relief)
This is an advertisement. Results vary based on individual debt profile, enrollment, and program terms. We are not a lender.
